Note

Note class Null safety

Note, i.e. a free-form string about the contact.

Android allows multiple notes per contact, while iOS supports only one.

On iOS13+ notes are no longer supported, as you need to request entitlement from Apple to use it in your app. See: https://developer.apple.com/documentation/bundleresources/entitlements/com_apple_developer_contacts_notes
